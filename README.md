### Installation
1. Clone this repository:
   ```bash
   git clone https://github.com/alaazamil123/ZAMILPRO.git
   ```

2. Navigate to the project directory:
   ```bash
   cd ZAMILPRO
   ```

3. Install the necessary dependencies:
   ```bash
   pip install -r requirements.txt
   ```
